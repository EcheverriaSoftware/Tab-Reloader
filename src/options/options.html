Tab Reloader

Settings & defaults

Default interval

New tabs use this interval unless you set a per-tab override. This value persists across browser sessions and may sync across your devices.

Allowed range: 1–1440 minutes (24 hours). Values outside are clamped.

Toolbar badge

Keyboard shortcut

Quick-add (or toggle) the current tab on the reload list: Alt+Shift+R

Opens the browser's extension-shortcuts page where you can rebind it.

How it works

  • The list of reloading tabs is per session — it clears when the browser closes.
  • A refresh is skipped silently while you're actively viewing that tab.
  • Manually reloading a tab resets its timer to a full interval.
